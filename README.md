# Next.js 시작하기
Nomad Coders의 [Next.js 시작하기 강의]를 기반으로 만든 영화 정보 탐색 웹사이트입니다.  
Next.js를 사용하여 라우팅, 데이터 패칭 방식, 로딩/에러 처리, 메타데이터 관리 등 Next.js의 핵심 개념을 학습하고 실습하는 것을 목표로 했습니다.

## 🔗 배포 링크
https://nextjs-movies-seven-nu.vercel.app/

## 🛠 사용 스택
- Framework: Next.js (App Router)
- Language: TypeScript
- UI: React
- Styling: CSS
- Deployment: Vercel
- Data Source: Nomad Coders에서 제공한 영화 API

## ⚙ 주요 기능
- 영화 목록 조회
- 영화 상세 페이지
  - 영화 제목, 별점, 개요 정보 표시
  - 공식 홈페이지로 이동할 수 있는 버튼
  - 트레일러 목록 제공
  - 코드 챌린지:
    - 출연진 목록 페이지
    - 제공자 목록 페이지
    - 유사한 영화 추천 페이지
- 동적 라우팅
- 로딩 / 에러 화면 처리
